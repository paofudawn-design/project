这个文件夹用于保存从 Notion 页面导出的 Markdown 文件。

## 导出选项

### 选项1：手动导出（推荐）
1. 在 Chrome 中打开 Notion 页面
2. 点击右上角 "..." 菜单
3. 选择 "Export"
4. 选择 "Markdown & CSV" 格式
5. 保存到本文件夹

### 选项2：公开分享
1. 在 Notion 中点击 "Share" 按钮
2. 选择 "Publish to web"
3. 获取公开链接
4. 系统可以自动抓取内容

### 选项3：修复浏览器控制服务
需要解决 Chrome DevTools 连接问题。
